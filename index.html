O acesso a páginas WEB via internet acontece através do relacionamento Cliente-Servidor.


Relação Cliente-Servidor

  • Cliente digita o endereço de uma página no navegador
  • O navegador busca pelo endereço da página em servidores
  • As páginas são pedidas e enviadas por protocolos TCP/IP
  • O cliente recebe a página buscada
